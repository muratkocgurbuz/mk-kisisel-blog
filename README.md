# Kişisel Blog Sitesi

Bu proje, React + Tailwind CSS ile hazırlanmış kişisel blog şablonudur.

## Özellikler
- Hakkımda bölümü
- Projeler listesi
- Resim & video alanı
- İletişim bilgileri
- Footer: © 2025 Murat Koçgürbüz - Öğrenci No: 2221032823

## Kurulum
```bash
npm install
npm start
```

## Yayınlama
```bash
npm run build
npm run deploy
```
